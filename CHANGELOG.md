# Changelog

## v1.5 (build 9) — 2026-05-15

### 새 기능
- **앱 내 메일 문의** — 앱 정보 → 이메일 문의가 앱 내 메일 작성기로 동작. 앱 버전 / iOS 버전 / 기기 모델 / 접근 권한이 본문에 자동 첨부되어 버그 보고가 진단 정보와 함께 도착. 메일 미설정 기기는 안내 후 신고 폼으로 안내.
- **심볼 모드 롱프레스 대체 문자** — 심볼 키패드에서 길게 누르면 대체 기호: `$`→₩ € £ ¥ / `*`→※ ★ ☆ / `(` `)`→[ ] { } < > / `-`→– — • / `/`→\\ / `:`→;
- **우측 컬럼 특수키 슬롯 독립화** — 모던 `#` 자리·확장형 1번 셀의 특수키 슬롯이 스페이스 옆 특수키와 별개로 편집됨. 한쪽을 바꿔도 다른 쪽이 따라 바뀌지 않음.

### UX 개선
- **레이아웃 설정 용어 정리** — "긋기 펑크" → "특수키", "슬롯 A/B/C" → "우측 컬럼 / 좌측 컬럼 / 스페이스 옆 키". 셀에서 내부 row 번호 노출 제거, 빨강 "초기화" → "되돌리기".
- **확장형 우측 컬럼 1번 셀** — `#` 고정에서 사용자 편집 슬롯 그대로 반영 (클래식과 공유).
- **한글 펑크 OFF** — 라디오/슬롯 편집기 자동 숨김 + 백스페이스 폭 2셀 정렬로 우측 라인 유지.
- **'전체 접근 허용' 안내** — 햅틱 설정 하단에 권한 안내 + iOS 설정 딥링크. 진동이 안 울리는 이유를 앱 내에서 바로 안내.

### 버그 수정
- **콜드 스타트 깜빡임 제거** — 키보드 첫 등장 시 (1) 시스템 그레이→설정 색, (2) 기본→사용자 레이아웃, (3) hosting view reattach 3중 flicker 제거. 설정을 SwiftUI 호스팅 이전에 로드하고, 첫 등장에는 touch-recovery 토글을 스킵.
- **무조건 클릭음 제거** — `clickSoundEnabled`를 무시하고 키보드 표시 때마다 울리던 오디오 워밍업 클릭 제거.

### 제스처
- **긋기 길이 ↔ 반전 임계값 연동** — "긋기 길이 길게" 설정이 정상 방향뿐 아니라 반전(↑→↓, ←→→) 임계값까지 함께 스케일. ㅚ/ㅞ 등 멀티 스트로크 모음 인식이 swipe length preset 에 일관되게 비례.

### 권한
- **전체 접근(Full Access) 선택적 사용** — 햅틱 진동을 위해 `RequestsOpenAccess` 활성화. 기본 입력은 권한 없이 동작하며, 키 입력은 외부로 전송되지 않음 (햅틱·사운드·앱 설정 동기화에만 사용).

### 알려진 이슈
- **지구본 키 연속 전환 시 깜빡임 누적 (iOS 시스템 제약)** — 다른 키보드 ↔ 모아+ 를 지구본 키로 *연속해서* 여러 번 전환하면, 전환 순간의 깜빡이는 회색 영역 크기가 점점 커집니다. 키보드는 매번 정상 위치/크기로 복귀하므로 입력은 정상 동작하며, 일반 사용(1~2회 전환)에서는 거의 체감되지 않습니다. iOS 가 키보드 전환 시 시스템 입력 컨테이너 frame 을 직접 누적 조작하는 OS 동작으로, 익스텐션 코드로 해결 불가함이 다각도 검증으로 확인되었습니다 (확인 환경: iOS 26.4, 이전 버전 미검증 — 상세: [docs/KNOWN_ISSUES.md](docs/KNOWN_ISSUES.md)). Apple Feedback 제출 예정.

## v1.4 (build 7) — 2026-05-09

### 새 기능
- **레이아웃 커스터마이즈** — 설정 → 키보드 → 레이아웃에서 모던 / 클래식 / 확장형 3가지 프리셋 중 선택. 첫 실행 시 모드 선택 모달.
- **슬롯 B 모음 키** — 스페이스 옆 키를 모음 입력 키로 변경 가능. 자음 키 드래그와 동일한 multi-stroke 모음 입력 (ㅑ ㅕ ㅛ ㅠ ㅒ ㅖ ㅢ ㅘ ㅙ ㅚ ㅝ ㅞ ㅟ 모두 입력).
- **A1 백스페이스 ↔ ㆍ swap** — 모던 모드에서 백스페이스 위치를 1.1 위치(아래)로 옮길 수 있는 토글.
- **A2/A3 우측 컬럼 사용자 매핑** — 클래식/확장형 모드의 col 6 키들을 자유 매핑.
- **슬롯 C 사용자 매핑** — col 0 (~^;*) 4 셀 자유 매핑.
- **마지막 모드 기억** — 한글/영문 마지막 사용 모드 유지 옵션 (설정 → 키보드 → 입력 동작).

### UX 개선
- **설정 IA 통일** — 메인 → 키보드 / 외형 / 반응 / 단축어 / 도움말 / 앱 정보 6 항목 평면 구조.
- **새 페이지** — KeyboardSettingsView, BackspaceSettingsView, InputBehaviorSettingsView, LongPressSettingsView, HelpView.
- **제스처 미리보기 위치** — 입력 좌표 반대편에 표시되어 손가락이 가리지 않음.
- **튜토리얼 건너뛰기** + **도움말 진입점**.
- **긋기 테스트 화면 개선** — 실제 키보드로 직접 입력 테스트 + 섹터 캔버스에 입력 결과 표시.

### 레이아웃 정리
- 영문/심볼 모드의 함수행 슬롯 B 키 제거 (스페이스가 흡수).
- 심볼 키패드 백스페이스 위치가 한글 백스페이스와 일치.
- 영문 shift / 백스페이스 1.5x 폭, 좌우 정렬.
- 함수행 토글 버튼 (123/한글) 폭이 좌측 col 0 키와 일치.
- 함수행 엔터 폭이 한글/영문 모두 동일.
- 클래식/확장형의 가로 백스페이스가 위 행과 정확히 정렬.
