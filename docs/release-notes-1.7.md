## 한국어 (App Store 제출용 — 권장)

```
이번 업데이트로 모아+가 iPad를 정식 지원하고, 입력은 더 정확해졌습니다.

📱 iPad 정식 지원
아이패드 전용 레이아웃과 가로·세로 분리 키보드를 지원합니다. 넓은 화면에서 더 편하게 입력하세요.

✨ 자음에서 바로 모음
자음 키를 ㅣ/ㅡ 방향(↗↖ / ↙↘)으로 긋고 이어서 그으면 모음이 완성됩니다. 예: ㄱ을 ↗→ 로 그으면 '가', ↙↑ 로 그으면 '고'.

🎯 입력 정확도 대폭 개선
• 8방향 좌·우 각도를 따로 조정
• 4방향 전용 모드 (대각선 끄기)
• 멀티스트로크 민감도 설정
• 긋기 테스트로 내 손에 맞게 미세 조정

작은 떨림이나 긴 진입 동작에도 후속 획을 더 정확히 인식합니다.

여러분의 피드백 덕분에 더 좋아졌습니다. 손끝으로 완성하는 한글, 모아+
```
