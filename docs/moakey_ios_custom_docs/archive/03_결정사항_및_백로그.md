# 결정사항 및 백로그

## 확정 결정사항

### 입력 구조
- 목표는 갤럭시 원본 양손 모아키 사용감 복원
- 우측 점 / ㅣ, 하단 ㅡ는 기호가 아니라 모음 primitive
- 긋기 각도와 긋기 길이는 별도 파라미터로 분리
- 긋기 길이는 `짧게 / 보통 / 길게` 3단계 enum

### 제스처 보정
- 전역 각도 1세트만으로는 해결 안 됨
- 각 자음 키 완전 개별 보정보다 먼저 세로 라인별 보정으로 구현
- 특히 끝열 바깥쪽 스와이프, ㅡ / ㅣ 인식 강화가 중요

### 보조 입력
- 각 키 롱프레스 시 숫자 / 일부 특수문자 즉시 입력
- 키 내부 작은 힌트 표시 필요
- 많은 특수문자는 별도 레이어로 분리
- 언어 변환 계열 키 짧게 탭으로 내부 특수문자 레이어 진입

### 약어 확장
- 예: `ㅎㅅㅁㅇ` -> `koh@move.kr`
- delimiter 확정 방식
- 치환 직후 backspace 1회로 원문 복원

### 외형
- 테마: 시스템 / 라이트 / 다크
- 배경: 이미지 + 투명도 조절
- 버튼 색상 테마 지원
- 햅틱 반응은 가능하면 추가

## 백로그

### P0
- 양손 모아키 레이아웃 복원
- 모음 primitive 처리
- 긋기 각도 / 길이 엔진
- 세로 라인별 제스처 보정
- 롱프레스 숫자 / 일부 특수문자
- 약어 확장

### P1
- 확장 특수문자 레이어
- 보조 힌트 크기 / 위치 조정
- 끝열 바깥쪽 스와이프 특화 보정
- 테스트 모드 / 디버그 오버레이

### P2
- 테마 / 배경 이미지 / 투명도
- 햅틱 / 클릭 사운드
- 버튼 색상 커스터마이징
