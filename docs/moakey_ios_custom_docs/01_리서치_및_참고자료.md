## 2) 외부 공개 자료
### 삼성 / 모아키
1. 삼성 서비스센터 안내  
   - 삼성 키보드에서 한국어 형식으로 한손 모아키 / 양손 모아키를 지원한다는 참고자료
2. 모아키 공식 Facebook 안내  
   - 2021년부터 Samsung Galaxy 기본 입력 모드로 추가, Play Store용 모아키 종료 공지
3. APKPure - org.samsung.app.MoAKey  
   - 기능 소개, 버전 히스토리, custom skin 관련 업데이트 흔적 확인
4. APKPure versions  
   - 1.6.221220 등 구버전 이력 확인용

### Apple / iOS custom keyboard
1. Configuring a custom keyboard interface  
   - 키보드 확장 기본 구조
2. Configuring open access for a custom keyboard  
   - 고급 기능 / shared container 관련 참고
3. App Groups / ExtensionScenarios  
   - 컨테이너 앱과 키보드 확장 간 설정 공유
4. playInputClick / UIInputViewAudioFeedback  
   - 표준 입력 클릭
5. UIFeedbackGenerator / UIImpactFeedbackGenerator  
   - 햅틱 피드백 적용 참고

## 3) 정리된 핵심 사실
- 갤럭시 삼성 키보드에는 양손 모아키가 실제 입력 형식으로 존재
- 기존 독립 모아키 앱/버전 자료가 남아 있어 복원용 힌트로 사용 가능
- 긋기 각도 / 긋기 길이 설정이 실제로 존재
- 긋기 길이는 `짧게 / 보통 / 길게` 3단계 enum
- iOS 커스텀 키보드는 컨테이너 앱 + 키보드 확장 구조로 가며, 공유 설정은 App Group 설계가 적합
- 표준 클릭 사운드와 UIKit 기반 햅틱 피드백 적용이 가능
