자음 키를 누르고 방향을 긋기만 하면 모음이 입력됩니다.
모아+는 모아키 방식의 제스처 한글 입력을 iOS에서 구현한 키보드 앱입니다.

[제스처 입력]
- 자음 키 위에서 8방향으로 긋기만 하면 21개 모음을 모두 입력
- 왕복 긋기로 야/여/요/유 계열 이중모음도 빠르게 입력
- 방향을 꺾으면 와/워/애/에 계열 복합 모음도 간단하게

[천지인 모음 입력]
- 세로획, 점, 가로획 세 키만으로 모든 한글 모음 조합
- 점 누적 입력 등 천지인 표준 합성 지원
- 자음 키 제스처와 자유롭게 섞어 쓸 수 있습니다

[영문 QWERTY 키보드]
- 한영 키로 한글과 영문을 즉시 전환
- Shift 더블탭 또는 길게 누르기로 Caps Lock
- 숫자 키 길게 누르면 특수문자 입력

[편집 도우미]
- 스페이스바를 좌우로 드래그하면 커서가 그만큼 이동
- 여는 괄호 입력 시 닫는 괄호 자동 삽입
- 백스페이스 길게 누르면 단어 단위 빠른 삭제

[롱프레스 보조 입력]
- 길게 누르면 숫자와 기호를 바로 입력
- 누른 채로 드래그하면 다른 후보를 선택
- 키별 매핑을 자유롭게 편집 가능
- 심볼 키패드에서 기호 키를 길게 누르면 통화 기호 등 대체 기호 입력

[약어 확장]
- 자음 몇 개만 입력하고 스페이스를 누르면 긴 문장으로 자동 치환
- 짧은 자음 조합으로 자주 쓰는 문구를 등록하면 입력 속도가 크게 향상

[나만의 키보드]
- 모던, 클래식, 확장형 세 가지 레이아웃 프리셋 중 선택 (첫 실행 시 선택)
- 스페이스 옆 키를 모음 입력 키로 전환 가능
- 우측 컬럼과 스페이스 옆 특수키를 각각 독립적으로 매핑
- 다섯 가지 색상 테마 프리셋 또는 직접 색상 선택
- 배경 이미지를 선택하고 원하는 영역을 크롭
- 키 배경 투명도를 조절하여 나만의 키보드 완성
- 긋기 각도, 길이, 대각선 매핑, 열별 보정을 한 화면에서 통합 관리
- 라이브 제스처 시각화 화면에서 실제 키보드와 똑같은 엔진으로 직접 시험

[타이핑 연습]
- 천지인, 영문 QWERTY, 커서 이동 등 33종 시나리오로 빠르게 익히기

[문의]
- 앱 정보 화면에서 바로 메일로 문의 (앱, iOS, 기기 정보가 자동 첨부)
- 메일 계정이 없어도 신고 폼으로 의견을 보낼 수 있습니다

[프라이버시]
- 네트워크를 사용하지 않습니다
- 입력 데이터를 수집하거나 저장하지 않습니다
- 기본 입력은 전체 접근(Full Access) 없이 동작합니다. 햅틱 진동을 쓰려면 전체 접근 허용이 필요하지만(선택), 켜더라도 키 입력은 외부로 전송되지 않습니다
